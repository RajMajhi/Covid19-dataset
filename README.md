## Project Overview

The goal of this project is to:

* Analyze COVID-19 data for India
* Perform data cleaning and preprocessing
* Generate meaningful visualizations
* Understand trends in cases, recoveries, and deaths

This project is implemented using **Jupyter Notebook** and Python data analysis libraries.

## How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Covid19-dataset.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Covid19-dataset
   ```

3. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

4. Open the Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

5. Run `Covid19 dataset.ipynb`
